zsh: command not found:nvm….解决办法

##解决方案
首先是在zsh未找到nvm，所以应该检查zsh的环境变量文件.zshrc。发现没有和nvm相关行。因为nvm的安装脚本把定义添加在了默认的bash的.bash_profile里，所以需要将引用copy到.zshrc

export NVM_DIR=”$HOME/.nvm”

[ -s “$NVM_DIR/nvm.sh” ] && . “$NVM_DIR/nvm.sh” # This loads nvm

[ -s “$NVM_DIR/bash_completion” ] && . “$NVM_DIR/bash_completion” # This loads nvm bash_completion

重启终端即可

##后续总结
其实这次会出现这问题的原因很简单，就是备考太久有点生了，不记得自己把bash换成zsh了。
